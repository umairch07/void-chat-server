# Chat App

A simple real-time room-based chat app built for fun and testing.

## What it does
- Enter a room code and username to join a chat room
- Anyone with the same room code shares the same room
- Supports text messages and image sharing
- No accounts, no message history, no logs

## Tech
- Frontend: HTML/CSS/JS — hosted on Netlify
- Backend: Node.js WebSocket server — hosted on Railway

## Note
This is a test project built for fun. Not intended for production use.
